Hello and welcome! Happy you're here, but there's nothing to see (yet). Check back in later, won't you? :) Meanwhile, read on if you want to learn more about this site.

I built this using VS Code with the Live Server Extension. Live Server helps to see the code changes in real time, so I don't spam the activity feed with updates. Plus, it's easier to code this way! (Maybe I'll make a page on the other extensions I use?)

Terminal info above references the neofetch command on Linux systems. Terminal theme is inspired by agnoster ohmyzsh theme.

Dark color scheme is based on Tokyo Night theme. Light colour scheme is not based on Tokyo Night; instead I handpicked different colours to make the transition better. If you want to see other colour schemes, check this out!

Here's a tutorial on how I built the colour scheme button. (Scroll down to see it.)

If you're interested in learning web development basics (HTML/CSS, JavaScript, etc), I encourage you to check out The Odin Project.
